Water Reminder Chatbot

Stay hydrated with this intelligent Water Reminder Chatbot, developed using Python. Designed as a personal wellness assistant, it sends timely water intake alerts through a conversational interface. The chatbot integrates time-based logic with simple natural language interactions, promoting healthy habits through automation. It's an ideal showcase of how AI-driven tools can enhance everyday routines with minimal user input.

Tech Stack:
  • Python
  • Tkinter
  • Time & Scheduling Modules
  • Rule-based Logic
  • Basic NLP Techniques
