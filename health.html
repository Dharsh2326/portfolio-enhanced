Mental Health Prediction Project

This data-driven mental health prediction system uses machine learning algorithms to identify patterns related to psychological well-being. Built using Python, the project includes preprocessing real-world data, training multiple classification models (Logistic Regression, Random Forest, SVM), and evaluating their effectiveness through robust metrics and visualizations. The project highlights the ethical and practical application of data science in understanding mental health trends and enabling early intervention.

Tech Stack:
  • Python
  • Pandas, NumPy
  • Scikit-learn
  • Matplotlib, Seaborn
  • Jupyter Notebook
